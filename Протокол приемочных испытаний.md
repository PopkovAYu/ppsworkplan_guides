| **Формулировка требования**                                                                                                                                                                     | **Комментарии**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| FR1. Доступ к генератору должен осуществляться по подтвержденной роли «сотрудник» на profile.tusur.ru                                                                                           | Генерация ИПРП доступна только для пользователей ФИО, должность и кафедра которых, совпадают с данными из АИС Мастер Планов и/или телефонным справочник (если создается ИПРП для сотрудника без плановой учебной нагрузки)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| FR2. В системе должна быть функция создания нового учебного года для инициации разработки ИПРП, доступная только учебному управлению                                                            | **-**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| FR3. Возможность генерации ИПРП на текущий учебный год преподавателем должна появляться только после инициации учебного года со стороны учебного управления                                     | -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| FR4. При генерации ИПРП вся необходимая информация, которая хранится в других системах ТУСУРа, должна подгружаться автоматически                                                                | К такой информации относятся:<br><br>- Документы ППС<br>    <br>- Учебная нагрузка ППС из АИС Мастер планов<br>    <br>- Личная информация ППС и заведующего кафедрой                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| FR5. В состав АИС должен быть включен справочник типов работ, который заполняется сотрудниками УУ                                                                                               | Справочник заполняется в соответствии с нормами времени                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| FR6. ИПРП должны разрабатываться для каждого учебного года независимо                                                                                                                           | Каждый учебный год это отдельная сущность, внутри которой возможна генерация и редактирование данных независимо от других учебных лет                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| FR7. В АИС должна быть реализована система согласования ИПРП с возможностью отправки на согласование и возврата на доработку                                                                    | В цепочку согласования входят преподаватель и  заведующий кафедрой                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| FR8. В АИС должна быть возможность добавления сотрудника кафедры, не являющегося ППС, с возможностью просмотра, редактирования, создания и отправки на согласования ИПРП преподавателей кафедры | В качестве такого сотрудника обычно выступает делопроизводитель/секретарь/методист кафедры. Данный сотрудник должен иметь возможность как создавать новый, так и редактировать существующий ИПРП и отправлять его на согласование                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| FR9. В АИС при согласовании ИПРП должна добавляться ПЭП лица, согласовавшего ИПРП, а к документу должен крепиться электронный сертификат ПЭП заведующего кафедрой                               | Подписание должно происходить на каждом этапе первичного и финального согласований – в момент отправки на согласование, в момент согласования. При этом, если ИПРП возвращен на доработку, отредактирован и снова отправлен на согласование, то крепится новая ПЭП                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| FR10. В АИС должна быть система выделения изменений в случае их внесения после отправки на согласование                                                                                         | В случае, если после отправки ИПРП на согласование в него внесены правки заведующим кафедрой или сотрудником кафедры, в цепочку согласования должно быть добавлено согласование преподавателем с заменой ПЭП, а все изменения, которые внесены должны быть выделены на каждой итерации согласования.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| FR11. В системе согласования должно быть поле с комментариями для указания необходимых доработок при возврате на доработку и отправке на согласование                                           | Поле не обязательно для заполнения                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| FR12. В системе согласования должны быть поля для ввода дат и номеров протоколов заседаний кафедры, на которых утверждаются ИПРП и результаты                                                   | Поле только для заведующего кафедрой                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| FR13. В АИС должна быть предусмотрена функция скачивания как полностью согласованного документа в формате PDF, так и возможность скачать документ в формате бланка                              | Документ в формате бланка должен быть полностью заполнен за исключением полей для подписей и дат, номеров протоколов заседаний кафедр, и страницы с заключением. Также в бланке должны быть дополнительные строки в таблицах с научной, методической и воспитательной работой для возможности добавления иных видов работ                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| FR14. В АИС должна быть система статусов ИПРП                                                                                                                                                   | Предусмотрены следующие статусы:<br><br>- Создан<br>    <br>- На согласовании у заведующего кафедрой<br>    <br>- Первично согласован<br>    <br>- Промежуточно согласован<br>    <br>- Финально согласован<br>    <br>- На доработке<br>    <br>- На согласовании у преподавателя<br>    <br>- Удален                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| FR15. В АИС должна быть система статистики статусов ИПРП                                                                                                                                        | -                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| FR16. В АИС должна быть система оповещений изменений в статусе ИПРП                                                                                                                             | Оповещения отображаются в виде счетчиков на вкладках соответствующего года обучения                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| FR17. После согласования ИПРП недоступен для редактирования в части удаления согласованных видов работы, однако может пополняться другими видами                                                | ИПРП может редактироваться преподавателем в любое время в части добавления других работ или внесения  результатов выполнения согласованных работ. Измененный файл отправляется на согласование заведующему кафедрой в любое время, однако «доходит» до него в строго фиксированные периоды в течение года (за 2 недели до и 2 недели после окончания осеннего семестра, за 2 и 2 недели месяца до окончания учебного года (сроки предварительные и могут меняться)). До того, как ИПРП «дошел» до заведующего кафедрой, преподаватель имеет возможность отзыва с согласования для внесения изменений и повторной отправки. Согласование в периоды после первого и перед последним не добавляют и не меняют ПЭП. ПЭП меняется только при первом согласовании и добавляется в момент финального согласования. |
| FR18. В ИПРП должна автоматически добавляться методическая работа, зависящая от учебной нагрузки                                                                                                | В нормах времени предусматривается методическая нагрузка, которая зависит от объема учебной нагрузки. Данные виды работ должны добавляться автоматически (подготовка к занятиям, проверка самостоятельной работы и т.д.)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| FR19. В АИС должны отслеживаться нормы времени методической, научной и воспитательных работ                                                                                                     | Данные часы при суммировании с учебной нагрузкой не должны превышать годовой бюджет времени, пропорциональный занимаемой ставке. В случае превышения нагрузки система должна выдавать предупреждение и прикладывать к файлу согласие на выполнение объема работ, превышающего установленные пределы.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
